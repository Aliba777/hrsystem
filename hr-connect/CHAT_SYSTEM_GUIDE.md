# 💬 Система чата в реальном времени - HR Connect

## 🎯 Возможности

### ✅ Реализованный функционал:
1. **Чат в реальном времени** (Long Polling - обновление каждые 3 секунды)
2. **Список всех бесед** с показом последнего сообщения
3. **Счетчик непрочитанных сообщений** в navbar (обновляется каждые 10 сек)
4. **Загрузка файлов** (PDF, DOC, DOCX, JPG, PNG) до 5MB
5. **Автоматическое создание чата** при принятии оффера
6. **Отметка прочитанных сообщений**
7. **Красивый UI** с градиентами и анимациями
8. **История сообщений** (хранится вся переписка)
9. **🆕 Реакции на сообщения** (👍 ❤️ 😂 😮 👏 🔥)
10. **🆕 Редактирование сообщений** (свои сообщения)
11. **🆕 Удаление сообщений** (мягкое удаление)
12. **🆕 Статус "онлайн/оффлайн"** собеседника
13. **🆕 Индикатор "печатает..."** в реальном времени
14. **🆕 Кнопки действий** при наведении на сообщение



## 📦 Установка

### 1. Создайте таблицы в базе данных:

**Вариант А (Рекомендуется - Автоматически):**
1. Откройте в браузере: `http://localhost/hr-connect/config/install_chat_features.php`
2. Дождитесь сообщения "Установка завершена успешно!"
3. Удалите файл `config/install_chat_features.php` для безопасности

**Вариант Б (Вручную через SQL):**
```bash
# Основные таблицы чата
mysql -u your_user -p your_database < config/create_chat_tables.sql

# Дополнительные таблицы для новых функций
mysql -u your_user -p your_database < config/update_chat_tables.sql
```

Или выполните SQL вручную через phpMyAdmin.

### 2. Структура файлов:
```
hr-connect/
├── messages.php              # Список всех бесед
├── chat.php                  # Окно чата
├── ajax/
│   ├── get_messages.php      # Получение новых сообщений
│   ├── send_message.php      # Отправка сообщения
│   └── get_unread_count.php  # Счетчик непрочитанных
├── uploads/
│   └── chat/                 # Папка для файлов чата
│       └── .htaccess         # Защита файлов
└── config/
    └── create_chat_tables.sql # SQL для создания таблиц
```

### 3. Права доступа:
Убедитесь что папка `uploads/chat/` имеет права на запись:
```bash
chmod 777 uploads/chat/
```

## 🔄 Как работает

### Автоматическое создание чата:
1. HR отправляет оффер кандидату
2. Кандидат принимает оффер
3. Автоматически создается беседа в таблице `conversations`
4. Обе стороны видят кнопку "Чатқа өту"

### Long Polling:
- JavaScript каждые **3 секунды** проверяет новые сообщения
- Счетчик непрочитанных обновляется каждые **10 секунд**
- Нет нагрузки на сервер, работает на любом хостинге

### Безопасность:
- Проверка прав доступа к каждой беседе
- Защита от SQL-инъекций (prepared statements)
- Валидация типов файлов
- Ограничение размера файлов (5MB)
- .htaccess защита папки с файлами

## 🎨 UI компоненты

### Список бесед (`messages.php`):
- Показывает все активные беседы
- Последнее сообщение

### Окно чата (`chat.php`):

- Автопрокрутка вниз

### Navbar:
- Иконка "Хабарламалар" с бейджем
- Красный счетчик непрочитанных
- Автообновление счетчика

## 📊 База данных

### Таблица `conversations`:
- `id` - ID беседы
- `offer_id` - Связь с оффером (UNIQUE)
- `hr_id` - ID HR
- `jobseeker_id` - ID соискателя
- `created_at` - Дата создания
- `last_message_at` - Время последнего сообщения

### Таблица `messages`:
- `id` - ID сообщения
- `conversation_id` - ID беседы
- `sender_id` - Отправитель
- `message` - Текст сообщения
- `file_path` - Путь к файлу (если есть)
- `file_name` - Имя файла
- `is_read` - Прочитано (0/1)
- `created_at` - Время отправки

## 🎨 Новые функции

### Реакции на сообщения:
- Наведите на сообщение
- Нажмите на иконку смайлика 😊
- Выберите реакцию: 👍 ❤️ 😂 😮 👏 🔥
- Повторный клик на ту же реакцию удалит её
- Видны все реакции с количеством

### Редактирование сообщений:
- Наведите на своё сообщение
- Нажмите иконку карандаша ✏️
- Введите новый текст
- Сообщение помечается как "(өңделген)"

### Удаление сообщений:
- Наведите на своё сообщение
- Нажмите иконку корзины 🗑️
- Подтвердите удаление
- Сообщение заменяется на "[Хабарлама жойылды]"

### Статус онлайн:
- Зеленый бейдж "Онлайн" если собеседник активен
- "Соңғы рет: X мин бұрын" если оффлайн
- Обновляется автоматически каждые 30 сек

### Индикатор "печатает...":
- Появляется когда собеседник набирает текст
- Исчезает через 3 секунды после остановки
- Работает в реальном времени

## 🚀 Использование

### Для соискателя:

2. Принять оффер

## ⚙️ Настройки

### Изменить интервал обновления:
В `chat.php` найдите:
```javascript
pollingInterval = setInterval(loadNewMessages, 3000); // 3 секунды
```

В `navbar.php` найдите:
```javascript
setInterval(updateUnreadCount, 10000); // 10 секунд
```

### Изменить максимальный размер файла:
В `ajax/send_message.php` найдите:
```php
if ($_FILES['file']['size'] > 5 * 1024 * 1024) { // 5MB
```

### Добавить новые типы файлов:
В `ajax/send_message.php` найдите:
```php
$allowed_extensions = ['pdf', 'doc', 'docx', 'jpg', 'jpeg', 'png', 'txt'];
```

## 🐛 Решение проблем

### Сообщения не обновляются:
- Проверьте консоль браузера (F12)
- Убедитесь что AJAX запросы проходят
- Проверьте права доступа к файлам

### Файлы не загружаются:
- Проверьте права на папку `uploads/chat/` (777)
- Проверьте размер файла (макс 5MB)
- Проверьте тип файла

### Счетчик не обновляется:
- Проверьте путь к `ajax/get_unread_count.php`
- Проверьте консоль браузера
- Убедитесь что пользователь авторизован
